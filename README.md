# 📌 Project Overview

Customer churn is one of the biggest challenges in the banking industry. Losing valuable customers directly impacts revenue and increases customer acquisition costs.

This project leverages Machine Learning to identify customers who are at risk of leaving the bank. By analyzing customer information such as credit score, account balance, geography, salary, and engagement, the model provides real-time churn predictions that help organizations improve customer retention strategies.

The application offers a modern dashboard with predictive analytics, customer insights, and interactive visualizations.

# 📊 Dataset

**Dataset Used**

Bank Customer Churn (Churn Modelling Dataset)

The dataset contains customer demographic, financial, and account-related information collected from a banking environment.

### Features Include

- Credit Score
- Geography
- Gender
- Age
- Tenure
- Account Balance
- Number of Products
- Credit Card Status
- Active Membership
- Estimated Salary

### Target Variable

- Customer Churn (Exited)
